# SlurmUI
Simple terminal UI for slurm

## Usage
```
pip install slurmui
```
To start, use type `slurmui`

## Control
| Shortcut | Description               |
|----------|---------------------------|
|d | delete job (Confirm with <<Enter>>) |
|l | see log|
|r | refresh |
|q/<Escape> | Abort|
|Arrow keys | cursor up/down |
|<Ctrl+C> | Quit |
